Appweb Quick Start

  1. Download

    First download Appweb to your system. You can download a binary package from http://appwebserver.org/downloads/appweb/download.ejs.

  2. Installation

    Then install Appweb on your system by following the instructions relevant to your system.

  3. Read the doc

    There is a wealth of documentation to read. Try the Users Guide and Programmers Guide first.
